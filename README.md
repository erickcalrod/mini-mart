# mini-mart

website for mini-mart

# Prerequisites

- Node.js (v18+ recommended)
- npm

# Local Setup

- git clone [Repo URL](https://github.com/erickcalrod/mini-mart.git)
- cd mini-mart

## Backend Setup

- cd backend
- npm install
- npm run dev
- Backend runs on http://localhost:3000

## Frontend Setup

- cd frontend
- npm install
- Create .env file with "VITE_API_BASE_URL=http://localhost:3000/"
- npm start
- Frontend runs on http://localhost:5173/
